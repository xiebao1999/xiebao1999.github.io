## 初始化仓库

在本地新建一个 Git 仓库：

```sh
git init
```

## 克隆仓库

从远程仓库克隆一个项目：

```sh
git clone <repository_url>
```

## 基本操作

### 查看状态

查看当前工作区状态：

```
git status
```

### 添加文件
将文件添加到暂存区：

```
git add <file_name>
```

添加所有文件：

```
git add .
```

### 提交更改
提交暂存区的更改：

```
git commit -m "commit message"
```
